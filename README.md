# 💰 Personal Finance Dashboard

> A browser-based personal finance tracker — manage income, expenses, budgets & savings goals with smart insights, charts, and PWA offline support.

🌐 **Live Demo:** [finova-budgethub.netlify.app](https://finova-budgethub.netlify.app)

---

## 🔴 Problem

Most people have no idea where their money goes every month. Existing apps are either too complex, require account signups, or send data to external servers — raising privacy concerns.

## ✅ Solution

A simple, private, browser-based tool — no login, no server, no setup. Everything stays in your own browser. Just open and start tracking.

---

## Features

- 📊 **Dashboard Overview** — Total income, expenses, net balance & savings rate
- 📈 **Interactive Charts** — Income vs Expense, category breakdown, monthly trends, savings progress
- 🏆 **Financial Health Score** — Smart score out of 100 based on your finances
- 💡 **Smart Insights** — Automated tips and warnings based on spending patterns
- 🎯 **Budget Limits** — Set per-category budgets with visual progress bars
- 💾 **Savings Goals** — Track multiple goals with progress and deadlines
- 🔔 **Bill Reminders** — Never miss a payment with due-date alerts
- 📅 **Monthly Summary** — Month-by-month income, expense & savings table
- 🧮 **EMI Calculator** — Calculate loan EMIs instantly
- 📤 **Export / Import** — Download or upload data as CSV
- 🌙 **Dark / Light Theme** — Toggle anytime
- 📱 **PWA Support** — Install as an app, works offline
- 🔒 **100% Private** — All data stored locally in your browser, no server

## Getting Started

No installation needed. Just open `index.html` in any browser.

```bash
git clone https://github.com/Dimshijain/Finance-Dashboard.git
cd Finance-Dashboard
# Open index.html in your browser
```

Or visit the **[Live Demo](https://finova-budgethub.netlify.app)** directly.
